## 📈 技術仕様

### コード規模
- **総行数**: 約5,200行（`apps/glb/index.html`）
- **関数数**: 50+関数
- **機能数**: 56機能

### データ構造
- **辞書インデックス**: `Map<string, string>` (150k語)
- **ユーザー辞書**: `Map<string, string>`
- **翻訳メモリ**: `Map<string, { targetValue, timestamp }>`
- **履歴**: `Array<{ source, target, fromLang, toLang, timestamp }>`
- **APIキャッシュ**: L1 (50件) + L2 (100件)

### パフォーマンス
- **辞書読み込み**: チャンク処理（UIブロックなし）
- **API削減**: Exact Cache + Cooldown + Rate Limiting
- **Trie検索**: 高速オートコンプリート
- **GPU加速**: CSS `will-change` + `transform: translateZ(0)`
