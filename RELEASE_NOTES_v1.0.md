# GLB v1.0 リリースノート

## 🚀 GLB v1.0 正式リリース

**リリース日**: 2026年1月

**GLB (Global Language Breaker)** は、旅行者のための統合翻訳・旅行サポートアプリです。

---

## ✨ 主要機能

### 1. 翻訳機能
- **150k辞書**: 15万語のオフライン辞書で高速翻訳
- **AI翻訳**: Gemini 1.5 Flashによる高精度翻訳
- **25言語対応**: 日本語、英語、中国語、韓国語など25言語をサポート
- **カメラ翻訳**: OCR機能付きカメラ翻訳
- **音声入出力**: 話すだけで翻訳、翻訳結果も音声で聞けます

### 2. 旅行特化機能
- **時差計算**: 文脈理解型時差アシスタント
- **チップ計算**: 状況に応じた最適化提案
- **電圧・プラグ情報**: 国別電圧・プラグ情報
- **文化・マナー情報**: 国別文化情報とタブー

### 3. 安全サポート
- **緊急連絡先**: 国別緊急連絡先（警察、救急、大使館）
- **緊急フレーズ集**: 8つの緊急フレーズを25言語で（オフライン対応）
- **安全情報**: 旅行安全のための8つのTips
- **医療情報**: 病院・薬局の探し方、慢性疾患の管理

### 4. 高度な機能
- **シーンガイド**: 状況別ガイド説明（オン/オフ切り替え可能）
- **単語説明**: 翻訳された単語をタップして説明を表示
- **リアルタイム翻訳**: 入力と同時に翻訳
- **会話モード**: 文脈を保持した会話翻訳
- **履歴管理**: 翻訳履歴の保存・管理
- **お気に入り**: よく使う翻訳を保存

### 5. 感染力・依存性強化
- **シェア機能**: Twitter/Facebook/LINE/コピー
- **紹介コードシステム**: 友達紹介で特典
- **毎日ログインボーナス**: 連続ログイン日数表示
- **使用統計表示**: 自分がどれだけ使ったか見せる

## 🔧 技術仕様

### フロントエンド
- **言語**: HTML/CSS/JavaScript
- **ホスティング**: GitHub Pages
- **OCR**: Tesseract.js
- **音声**: Web Speech API

### バックエンド
- **API**: NEゲートウェイ（Gemini 1.5 Flash）
- **APIコスト**: 0.025円/回
- **データ永続化**: Local Storage（GLBシリーズ標準データ構造）

### パフォーマンス
- **チャンク読み込み**: 150k辞書を効率的に読み込み
- **GPU加速**: アニメーションの最適化
- **API削減ロジック**: キャッシュ、クールダウン、レート制限
